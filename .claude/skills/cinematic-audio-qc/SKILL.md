# Cinematic Audio QC Skill

Use this skill before accepting any game music or sound asset.

## QC Checklist

Check:

1. originality
2. gameplay function
3. loop cleanliness
4. BPM consistency
5. bar alignment
6. downbeat clarity
7. section clarity
8. stem compatibility
9. dynamic range
10. loudness
11. clipping
12. harsh frequencies
13. muddy low end
14. transition usability
15. metadata completeness
16. file naming
17. license/source notes

## Red Flags

Reject or revise if:

- it sounds like a copied song
- the loop click/pops
- the loop is not bar-aligned
- BPM is unknown
- no metadata exists
- stems are not synchronized
- stingers do not land cleanly
- audio is too repetitive
- audio distracts from gameplay
- file names are vague
- source/license notes are missing

## Required Output

For each asset, produce:

- Pass/Revise/Reject
- problems found
- exact fixes
- metadata status
- implementation readiness
- next action
